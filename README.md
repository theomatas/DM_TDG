# DM TDG

Bienvenue dans le projet Théorie des graphes, EFREI 2018.
Créé par Théo Matas, Adrien Albanese, Rémi Cabo-broseta, Vincent Letourneur.
Professeur en charge : Hervé Barbot

Depuis le dossier principal, aller dans le dossier Src.
Vous y trouverez la fonction main qu'il faudra lancer avec python 3.7.
La fonction main une fois lancée ira chercher les graphes présents dans le dossier graph,
les parssera et les stockera en mémoire grace à une instanciation de la classe Graph.

L'utilisateur pourra ensuite choisir un graphe pour effectuer les différentes opérations demandées.
Les opérations sont groupées par exercices et l'utilisateur aura juste à presser la touche ENTRER pour les faire défiler.
Si l'utilisateur dispose de la bibliothèque tkinter, le script proposera à ce dernier de l'activer s'il le souhaite pour afficher les graphes (sans circuit).
Si le script n'arrive pas à ouvrir un fichier où accéder à l'interface cela ne l'empêchera pas de fonctionner.
